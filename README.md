# Advanced Python Calculator

A comprehensive, command-line calculator built in Python utilizing Object-Oriented Programming (OOP) principles. This project showcases clean code structure, error handling, and a variety of mathematical operations ranging from basic arithmetic to advanced scientific functions.

##  Key Features

* **Object-Oriented Design:** Implements **Single Inheritance** to cleanly separate the core calculator logic from the user interface.
* **15 Math Functions:** Supports everything from basic operations to advanced calculations using Python's built-in `math` module.
* **Calculation History:** Uses a **List** to dynamically store and view previous calculations during the session.
* **Continuous Execution:** Employs a `while` loop combined with `if-elif-else` control flow to allow uninterrupted multiple calculations until the user chooses to exit.

##  Concepts Demonstrated

* **Object-Oriented Programming:** Single Inheritance, Classes, and Methods.
* **Control Flow:** `while` loops for application lifecycle, `if-elif-else` for menu selection.

* **Standard Library:** Deep integration with Python's native `math` module.
